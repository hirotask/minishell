# minishell

A minimal shell.

## Build & Run

Run the following commands:

```
meson build/
meson compile -C build/
build/minishell
```

## Contributing

### VS Code

If you use [VS Code](https://code.visualstudio.com/), open this project in a devcontainer to start developing.

### NeoVim

If you want to use [NeoVim](https://neovim.io/), install the [devcontainer CLI](https://github.com/devcontainers/cli) first.
Then run the following commands to start developing.

```
devcontainer up
devcontainer exec bash
```
